# MaskedTextField

O MaskedTextField é um componente de Input que aceita uma máscara para formatar os valores

## Propriedades

<Props of={MaskedTextField} />

## Uso básico

<Playground>
  {() => {
    const [cep, setCep] = React.useState("36500032");
    const onChange = (e) => {
      setCep(e.target.value)
    };
    return (
      <MaskedTextField value={cep} mask="99999-999" onChange={onChange} variant="outlined"/>
    );
  }}

</Playground>
