# PasswordTextField

O PasswordTextField é um componente de Input que esconde ou exibe a senha de acordo com a ação do usuário

<Props of={PasswordTextField} />

## Uso básico

<Playground>
  {() => {
    const [senha, setSenha] = React.useState('');
    const onChange = e => {
      setSenha(e.target.value);
    };
    return <PasswordTextField onChange={onChange} value={senha} />;
  }}
</Playground>
