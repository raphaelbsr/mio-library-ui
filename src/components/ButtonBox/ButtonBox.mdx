# ButtonBox

ButtonBox é um container para organizar os Buttons.
Segundo a guideline do MIO os botões são sempre dispostos da direita pra esquerda

## Propriedades

<Props of={ButtonBox} />

## Uso Básico

<Playground>
  <ButtonBox buttonsWidth={150}>
    <Button variant="contained">Botão 1</Button>
    <Button variant="contained" color="primary">
      Botão 2
    </Button>
    <Button variant="contained" color="secondary">
      Botão 3
    </Button>
  </ButtonBox>
</Playground>
