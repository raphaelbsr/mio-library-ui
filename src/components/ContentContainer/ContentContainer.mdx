# ContentContainer

O ContentContainer é um elemento de layout básico baseado no [Box do Material-UI](https://material-ui.com/pt/components/box/).

Deve ser utilizado como primeiro componente da página de uma aplicação MIO.

## Propriedades

<Props of={ContentContainer} />

## Uso Básico

<Playground>
  <ContentContainer>
    <h1>Título</h1>
    <h2>Subtítulo</h2>
    <content>Conteúdo</content>
  </ContentContainer>

</Playground>
